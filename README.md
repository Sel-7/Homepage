# Homepage
Personal Homepage project for the Scrimba Module


I've utilised the teachings at scrimba to create a homepage using a wide variety of CSS techniques, the most notable being flexbox manipulation.
I've also tried to comply with the DRY coding practice with adding multiple classes into the same styling for example 

.guide-container, #guide-img {
    border-radius: 5px;
}

#hero-title, #hero-desc {
    padding: 6px 12px;
    border-radius: 10px;
}
